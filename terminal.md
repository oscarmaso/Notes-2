# TERMINAL

## Redirecciones

* &gt; : redireccion de salida. Crea un fichero nuevo.
* &gt;&gt; : doble redireccion de salida anexa fichero.
* &gt; : redirigir la entrada.
* &gt;&gt; : here document.
* &gt;&gt;&gt; : here string.


## Metacaracteres

* ~   : Mi direccion home
* .   : Directorio actual
* ..  : El cirectorio de arriba
* -   : El fichero que representa la terminal.
